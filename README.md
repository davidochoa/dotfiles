## Description

These dotfiles configure my shell, and help me get a new system up and running with my personal configuration/preferences.

### 1. Install

```
git clone https://github.com/davidochoa/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
setup/init.sh
```
